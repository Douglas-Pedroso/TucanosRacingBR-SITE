🚀 COMO USAR:

   1️⃣  DESENVOLVIMENTO:

   3️⃣  DEPLOY NO GITHUB:

       $ git commit -m "Inicial"
       $ git push origin main

       https://seu-usuario.github.io/Tucanos/
